# InternPilot 项目进度与交接记录

最后更新：2026-07-10

## 项目目标

1. 完成《数据挖掘》课程期末作业，体现机器学习、深度学习和人工智能应用。
2. 建设一个真实可用、可以上线的个人实习求职工具，并作为简历项目展示。

后续功能设计与技术决策需要同时服务以上两个目标，不能只满足课程演示，也不能忽略课程对数据挖掘相关方法的要求。

## 协作与交接约定

- 每次任务都更新本文件，记录完成内容、验证结果、遗留问题和下一步。
- 需求变化与重要设计决策应形成独立文档，避免只存在于聊天记录中。
- 发现既有实现存在疑问或有改进空间时，先记录依据、影响和建议，再与项目所有者确认。
- 不提交 API Key、Token、Cookie、真实简历、私人联系方式或含真实个人数据的数据库。
